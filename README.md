# jControlEngine

jControlEngine includes:
   - CasADi
   - cvxpy
   - Cvxflow
   - JModelica
   - Python Control Systems Library

## Usage
You can run the container like this:

`docker run -p 8888:8888 mechatronics3d/jcontrolengine`

Then go to this address on your browser:

`http://localhost:8888/`
